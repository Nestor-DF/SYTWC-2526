# Short Videos Web App

Esta es una aplicación web sencilla inspirada en los formatos de vídeo corto populares como **TikTok**, **YouTube Shorts** e **Instagram Reels**.
El propósito del proyecto es **practicar y testear funcionalidades multimedia en la web**.

Además de las funcionalidades solicitadas originalmente, también se simuló el comportamiento de **scroll / swipe down** característico de las plataformas de vídeo corto, permitiendo navegar entre clips de forma fluida.

---

## Instalación y ejecución

1. Instalar dependencias:

```bash
npm i
```

2. Ejecutar el entorno de desarrollo:

```bash
npm run dev
```

3. Abrir la app en el navegador en:

```
http://localhost:5173
```

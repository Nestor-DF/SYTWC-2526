# League of Legends Skin Viewer

A lightweight 3D skin viewer for League of Legends built with React, Vite, and react-three-fiber.

---

## Requirements

* Node.js 18 or higher
* npm, pnpm, or yarn (examples use npm)

Check your Node version:

```
node -v
```

---

## Setup

Install dependencies:

```
npm install
```

---

## Development

Start the Vite dev server:

```
npm run dev
```

Open in your browser:

```
http://localhost:5173
```
